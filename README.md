Generic Plants & Fauna Co. Landing Page
----------------------------------------------------------------
This project is a rework of the simple responsive website created for a web a development assignment. The site now has more Advanced CSS.

The website represents a fictional company called Generic Plants & Fauna Co., which focuses on plants, nature, and environmental services.

Pages:
----------------------------------------------------------------
  *index.html – Main landing page

  *features.html – Features of the service

  *pricing.html – Example pricing plans

  *about.html – Information about the company

  *contact.html – Contact information

Technologies Used:
----------------------------------------------------------------
  *HTML

  *CSS

  *Flexbox

  *CSS Grid

  *Javascript
